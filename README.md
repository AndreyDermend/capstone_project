# Prototype Starter Package

This package includes:
- Normalized mutual + unilateral NDA data
- A master clause library for deterministic assembly
- Canonical questionnaire and placeholder mapping configs
- A starter `assemble_contract.py` function

## First prototype path
1. Review `config/questionnaire_schema.json`
2. Review `config/assembly_order.json`
3. Run `python app/assemble_contract.py`
4. Wrap the logic in a simple Streamlit UI
